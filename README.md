# Personal Portfolio

This is my personal portfolio website built with React and Vite. It showcases my skills, projects, and professional experience in Cloud Computing and Big Data.

## Features

- Modern and responsive design
- Blue and purple theme
- Social media links configured via environment variables
- Contact form
- Project showcase
- Skills and experience sections

## Tech Stack

- React 18
- Vite
- React Icons
- CSS3
- Docker
